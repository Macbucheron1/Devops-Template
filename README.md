## Install

Follow those step to install the project localy:

1. Clone the repository:

    ```bash
    git clone https://github.com/Macbucheron1/EceDevops_lab1.git
    ```

2. Access the project file :

    ```bash
    cd nom-du-projet
    ```

3. Install dependencies :

    - Avec **npm** :

      ```bash
      npm install
      ```

4. Start the application :

    - Avec **npm** :

      ```bash
      npm start
      ```
